🔗 API ENDPOINTS:
================================================================================

List All Competitions (Admin View)
POST /api/v1/admin/competitions
├─ Auth: Bearer token (Admin role)
├─ Body: Competition data + prizes + judges + sponsors
└─ Response: 201 Created

Get All Competitions
GET /api/v1/admin/competitions?status=published&sort_field=created_at
├─ Auth: Bearer token (Admin role)
├─ Filters: status, category_id, featured, search
├─ Sorting: sort_field, sort_direction
└─ Response: 200 OK (paginated)

Get Single Competition
GET /api/v1/admin/competitions/{id}
├─ Auth: Bearer token (Admin role)
├─ Params: Competition ID
└─ Response: 200 OK

Update Competition
PUT /api/v1/admin/competitions/{id}
├─ Auth: Bearer token (Admin role)
├─ Body: Updated fields
└─ Response: 200 OK

Delete Competition
DELETE /api/v1/admin/competitions/{id}
├─ Auth: Bearer token (Admin role)
├─ Behavior: Archives if submissions exist, deletes if empty
└─ Response: 200 OK
